stash
1) git stash (to save changes temporarily)
2) git stash pop (to apply stashed changes)
3) git stash drop (to delete stashed changes)
4) git stash list (to check stashed changes)
5) git stash clear (to clear all stashed changes)
6) git stash apply (to apply stashed changes without removing from stash list)
7) git stash show (to show stashed changes)
8) git stash save "message" (to save stashed changes with message)
9) git stash branch "branch name" (to create new branch from stashed changes)
10) git stash push -m "message" (to save stashed changes with message)
11) git stash push -k (to keep index when stashing)
22) git stash push -p (to stash only changes in the working directory)
13) git stash push -u (to stash untracked files)
14) git stash push -a (to stash all changes including untracked and ignored files)
15) git stash push -q (to suppress output when stashing)
16) git stash push -m "message" -k (to save stashed changes with message and keep index)
17) git stash push -m "message" -p (to save stashed changes with message and stash only changes in the working directory)
18) git stash push -m "message" -u (to save stashed changes with message and stash untracked files)
19) git stash push -m "message" -a (to save stashed changes with message and stash all changes including untracked and ignored files)
20) git stash push -m "message" -q (to save stashed changes with message and suppress output when stashing)
